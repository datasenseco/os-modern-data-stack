# Data Stack Project

Este projeto configura um stack open source de dados utilizando Docker e Docker Compose, incluindo as seguintes tecnologias:

- Apache Airflow
- Airbyte
- Minio
- DBT
- PostgreSQL
- DataHub
- Metabase

## Pré-requisitos

- Docker
- Docker Compose
